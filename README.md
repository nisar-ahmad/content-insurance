# Content Limit Insurance
A sample app to demonstrate Renter's Contents Limit Insurance.

Users can add and delete items. Each item has a value and is categorized. The app calculates total value of all items and also value per category.

## Technologies

1. ASP.NET Core 3.1
2. React
3. Redux (Actions, Reducers, Store)
4. Typescript
5. SQL Server (Local DB)
